# 🎭 VTuber Overlay Tool

Инструмент для VTuber-стриминга: отображает кастомного персонажа поверх фона, который реагирует на громкость микрофона. Совместим с OBS Studio и Discord.

---

## ⚡ Установка

### 1. Требования
- **Python 3.10+** (скачать: https://python.org)
- Windows 10/11 (для macOS/Linux — незначительные отличия)

### 2. Установка зависимостей
Открой терминал в папке проекта и выполни:

```bash
pip install -r requirements.txt
```

> ⚠️ Если `sounddevice` не устанавливается — установи сначала:
> `pip install sounddevice` или используй `conda install -c conda-forge python-sounddevice`

### 3. Запуск

```bash
python main.py
```

## 🎮 Как использовать

### Шаг 1 — Открой выходное окно
Нажми **"📺 Open Output Window"**. Это отдельное окно — именно его ты будешь захватывать в OBS или шарить в Discord.

### Шаг 2 — Настрой камеру
- Выбери камеру из списка
- Нажми **"▶ Start Camera"**
- Переключись в режим **📷 Camera** — в выходном окне появится изображение с камеры

### Шаг 3 — Настрой VTuber-режим

#### Фон
- Нажми **"Choose…"** рядом с "Background"
- Выбери любое изображение (JPG, PNG, WEBP)

#### Слои персонажа
Персонаж — это набор изображений (слоёв), каждый из которых показывается при определённой громкости голоса.

**Пример настройки:**

| Файл | Min | Max | Когда показывается |
|------|-----|-----|--------------------|
| `body.png` | 0.00 | 1.00 | Всегда |
| `mouth_closed.png` | 0.00 | 0.20 | В тишине |
| `mouth_open.png` | 0.20 | 0.70 | При разговоре |
| `mouth_wide.png` | 0.70 | 1.00 | При громком крике |

Нажми **"➕ Add"** и выбери файлы. Для каждого задай диапазон громкости (0 = тишина, 1 = максимум). Используй пресеты для удобства.

**Редактирование слоя:** двойной клик по слою в списке.

**Порядок слоёв:** кнопка **"▲"** поднимает слой выше (рендерится поверх предыдущих).

### Шаг 4 — Запусти микрофон
- Выбери микрофон из списка
- Нажми **"🎙 Start Microphone"**
- Проверь индикатор громкости — говори и следи за реакцией

**Чувствительность:** если персонаж не реагирует — увеличь ползунок.

### Шаг 5 — Переключись в VTuber-режим
Нажми **"🎭 VTuber"** — в выходном окне появится фон и персонаж.

---

## 🖱️ Управление персонажем в выходном окне

| Действие | Результат |
|----------|-----------|
| Перетащить | Переместить персонажа |
| Скролл колёсиком | Изменить размер |
| Ctrl + Скролл | Точная настройка размера |
| Перетащить уголок (синий квадрат) | Изменить размер |

---

## 😱 Эффект крика
При громкости выше порога **"😱 Scream at"** поверх персонажа появляется красное свечение. Порог настраивается ползунком (по умолчанию 85%).

---

## 📺 Настройка OBS Studio

1. Добавь источник: **"Захват окна"** (Window Capture)
2. Выбери: **"VTuber Output — Capture this window in OBS"**
3. Включи **"Захватывать курсор"** → Нет

**Совет:** Если фон зелёный — можно использовать хромакей. Или поставь любой PNG-фон с прозрачностью.

## 📁 Форматы изображений
- Фон: PNG, JPG, JPEG, BMP, WEBP
- Слои персонажа: PNG (рекомендуется — поддерживает прозрачность!), JPG, BMP, **SVG**

## 🐛 Решение проблем

| Проблема | Решение |
|----------|---------|
| Камера не найдена | Нажми 🔄 для повторного сканирования |
| Звук не работает | Проверь, что микрофон разрешён в системе |
| Персонаж не реагирует | Увеличь ползунок чувствительности |
| Лаги в выходном окне | Закрой другие приложения, уменьши разрешение |
| SVG не загружается | Установи: `pip install PyQt5` с модулем QtSvg |
